SUBPART 5334.2 – EARNED VALUE MANAGEMENT SYSTEM

5334.203 Solicitation provisions and contract clause

If an Earned Value Management System (EVMS) waiver is obtained in accordance with the Adaptive Acquisition Framework Document Identification Tool (AAFDID) EVMS Application Requirements Table, contracting officers may omit the use of EVMS provisions and clauses prescribed for use in DFARS 234.203. The CAE approved EVMS waiver must be included in the contract file.
